# Lab 3: Dynamic Programming (GUI)

Программа для решения задачи оптимального распределения средств между предприятиями.

## Возможности
- Ввод количества предприятий и бюджета
- Таблица прироста прибыли
- Решение методом динамического программирования
- Графический интерфейс (Tkinter)
